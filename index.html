Projects.

Listed below are some of the many projects that I have worked on over the course of my schooling.

App to track checked out books.

I worked with a team of two other students to create an app to simulate checking out books from a libary. We built this app to improve our knowdlge of using sql to get data from a database and learn android programming techniques. We used android studio for the programming of the app and sql to querry the database. My role was mainly on creating the interface for the app.

Etsy Like web app.

I worked with a team of 3 other students over the course of a semester to create a web app similar to etsy. We used ruby on rails to build a responsive and interactive web application. I worked mostly on the back end database development.

Game AI for pacman like game.

Created AI for a pacman like game. The code for it was written in c++ and used a variation of dijkstra algorithm for the pathfinding. And the code was efficent enough to run multiple entites of AI with no preformace impact.

Senior Design.

For my senior design project, I worked with a team of five other seniors to create a system and smartphone app to measuere the amount of nitrates in a water sample. My main role on the project was to create the app to control and process the data from the spectrometer that was sent over bluetooth. I learned so much about android programming over the duration of the project as I had not done much of it before. I also learned how bluetooth works and how to utlize it to transfer data. Here is a link to our project website http://may1719.sd.ece.iastate.edu/. This project has many uses, one of which is the ablity to test water samples for quality while out in the field as the device is very small and portable.

Reflection.

My time at ISU has greatly prepared me to enter the workforce. Many of my classes have taught things such as what design patterns work best for certian situations. And they have also done a great job at making me think and solve problems through the many problems that I have had to do. ISU has also done a great job at prepareing me to work in groups. Almost every class has some sort of group programming componet and I think it has done a great job preparing me for the future. ISU has also done a great job at helping me understand ethical issues by going over many famous ethics issues and the IEEE code of ethics. There were often times that I had to use information and resources that were not provided in class to help me complete my work. One example of this is building an android app for a project, I had to use androids documentation online to even get anywhere. The career fair really helped me understand the importance of Lifelong Learing, it did this by making me realize that there are so many areas to go into with software and they are all eveloving at a rapid pace so you have to keep up with everything new. If I were to do my undergraduate work again there isn't much I would change. Howerver one thing I would change, is that I would have taken SP CM 212 eairler. I think this class would have help prepare me to give presentations in class more. I also would have focused on more of the embedded programming side of things as I think that is very interesting.
