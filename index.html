Projects.

Listed below are some of the many projects that I have worked on over the course of my schooling.

App to track checked out books.

I worked with a team of two other students to create an app to simulate checking out books from a libary. We built this app to improve our knowdlge of using sql to get data from a database and learn android programming techniques. We used android studio for the programming of the app and sql to querry the database. My role was mainly on creating the interface for the app.

Etsy Like web app.

I worked with a team of 3 other students over the course of a semester to create a web app similar to etsy. We used ruby on rails to build a responsive and interactive web application. I worked mostly on the back end database development.

Game AI for pacman like game.

Created AI for a pacman like game. The code for it was written in c++ and used a variation of dijkstra algorithm for the pathfinding. And the code was efficent enough to run multiple entites of AI with no preformace impact.
